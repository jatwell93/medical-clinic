## Overview

A professional-grade site feasibility analysis (Colab notebook + executive PDF) evaluating whether 292-296 Johnston St, Abbotsford VIC 3067 can support an independently profitable 5-FTE-GP medical clinic alongside a Priceline Pharmacy. The verdict is binary GO/NO-GO, gated on steady-state EBITDA > 0 AND margin > 15%, with tornado zero-crossings as flip-conditions. Pharmacy synergy is order-gated AFTER the standalone verdict and labelled "Secondary upside — NOT Load-Bearing."

## Milestone Summary

**Key Decisions:**

- Verdict gate: Steady-state EBITDA > 0 AND margin > 15% (D-01) — payback is context, not the gate
- Binary GO/NO-GO (D-03) — no CONDITIONAL tier; investors want a clear answer
- Verdict is 100% standalone (D-04, Core Value) — verdict text never mentions pharmacy
- Pharmacy synergy order-gated AFTER standalone scenarios (FIN-07) — can never rescue an unprofitable clinic
- EPSG:7855 (GDA2020/MGA zone 55) for all buffer/area math — fixes v1's degree-buffering catchment flaw
- Transparent arithmetic demand model — no scikit-learn on <30 rows (v1's Random Forest was statistically meaningless)
- Places API (New) with field masking + filesystem caching — $0 API cost on re-runs
- Colab-only PDF via weasyprint (D-18) — local Windows produces HTML only
- clinic_pnl(a) as pure function of parameters dict (ARCHITECTURE.md Pattern 3) — enables cheap scenario overrides
- Service-fee % revenue model (Pitfall 11 guardrail) — practice retains 30-35% of GP billings, not gross billings
