2. **Code cell (§6.2 clinic_ramp_monthly function)** — implement `clinic_ramp_monthly(a: dict, months: int = 36) -> dict` EXACTLY as specified in 04-RESEARCH.md "Monthly Ramp Function Skeleton", with these concrete values:
   - `import numpy as np` at the top of the cell
   - `gp_milestones = a["gp_ramp_milestones"]` — dict `{0: 2, 6: 3, 12: 4, 18: 5}`
   - `book_fill_curve = a["book_fill_curve"]` — dict `{0: 0.40, 3: 0.50, 6: 0.60, 9: 0.68, 12: 0.75}`
   - Helper function `gp_start_month(gp_idx, milestones)`: returns the month this GP started. GP 0 and 1 start at month 0 (2 GPs at open), GP 2 starts at month 6, GP 3 at month 12, GP 4 at month 18. Derive from the milestones dict: sort milestone months, GP indices map to milestone entries in order.
   - Helper function `interpolate_book_fill(months_active, curve)`: linear interpolation between curve points. For `months_active < 0` return 0 (GP hasn't started). For `months_active >= 12` return `curve[12]` (0.75, steady state). Between points, linearly interpolate.
   - For each month `m` in `range(months)`:
     - `gp_fte = max(v for k, v in gp_milestones.items() if k <= m)` (GP count at this month)
     - For each active GP (index 0..gp_fte-1): compute `months_active = m - gp_start_month(gp_idx, gp_milestones)`, then `util = interpolate_book_fill(months_active, book_fill_curve)`, accumulate `total_util += util`
     - `avg_util = total_util / gp_fte if gp_fte > 0 else 0`
     - Build `ramp_overrides = {"n_gp_fte": gp_fte, "utilisation": avg_util, "nurse_fte": gp_fte * 0.2}` (D-07: nurse scales with GP; reception_fte and manager_fte stay at steady-state values — fixed from day 1)
     - `month_result = clinic_pnl({**a, **ramp_overrides})`
     - `monthly_revenue[m] = month_result["practice_revenue"] / 12`
     - `monthly_costs[m] = month_result["total_costs"] / 12`
     - `monthly_ebitda[m] = month_result["ebitda"] / 12`
   - `monthly_cashflow = monthly_ebitda.copy()`; `monthly_cashflow[0] -= a["fitout_total"]` (fit-out paid at opening, D-12)
   - `cumulative = np.cumsum(monthly_cashflow)` (Don't Hand-Roll — no manual loop)
   - `peak_capital_modelled = np.min(cumulative)` (deepest cash trough, negative)
   - `peak_capital_total = abs(peak_capital_modelled) + a["working_capital_buffer"]` (D-12: buffer added ON TOP)
   - `breakeven_operating = int(np.argmax(monthly_ebitda > 0)) if np.any(monthly_ebitda > 0) else None` (Don't Hand-Roll)
   - `breakeven_payback = int(np.argmax(cumulative >= 0)) if np.any(cumulative >= 0) else None` (Don't Hand-Roll)
   - Return dict with keys: `monthly_revenue`, `monthly_costs`, `monthly_ebitda`, `monthly_cashflow`, `cumulative`, `peak_capital_modelled` (= `abs(peak_capital_modelled)`), `peak_capital_total`, `breakeven_operating`, `breakeven_payback`, `fitout_total` (= `a["fitout_total"]`), `working_capital_buffer` (= `a["working_capital_buffer"]`)
   - Docstring must cite: D-05 (monthly 36-month), D-06 (milestones + book-fill), D-07 (variable vs fixed staffing), D-08 (both breakevens), D-12 (peak capital + buffer), D-16 (per-month clinic_pnl calls)
